Analisis Faktor Adopsi Kebijakan Hijau

Studi menggunakan regresi logistik pada 200 entitas untuk mengidentifikasi faktor dominan dalam adopsi kebijakan berkelanjutan

Kesimpulan Utama:

  • Teknologi hijau merupakan faktor paling dominan (OR = 1.822, p < 0.001)
  • Setiap penambahan 1 startup hijau meningkatkan peluang adopsi sebesar 12%
  • Model memiliki akurasi keseluruhan 85.5% dengan sensitivitas 92.1%
  • Nagelkerke R² = 0.628 menunjukkan model menjelaskan 62.8% variasi keputusan

Peramalan Produksi Energi Terbarukan

Pemodelan ARIMA untuk memproyeksikan produksi energi terbarukan Indonesia 2020-2030 berdasarkan data historis

Kesimpulan Utama:

  • Model ARIMA(4,0,1) terpilih berdasarkan AIC (5.075) dan R² (0.745)
  • Uji ADF (p=0.031) menunjukkan data stasioner pada level (d=0)
  • Produksi diproyeksikan stagnan di kisaran 107-109 TJ hingga 2030
  • Model memiliki akurasi tinggi dengan MAPE 5.53% dan Theil's U 0.03
  • Diperlukan intervensi kebijakan untuk mengatasi stagnasi produksi
